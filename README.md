# Bankuptcy-Prediction
This is the effort to build a model to predict bankruptcy based on labelled dataset of companies gone bankrupt in last 20 years.
# 📌 Bankruptcy Prediction — Case Study
### Problem
Financial institutions need early warning indicators to identify companies at risk of bankruptcy.
### Goal
Build a predictive model using financial ratios and historical data to classify bankruptcy risk.
### My Role
Defined problem framing, selected features, evaluated models, and interpreted results for business stakeholders.
### Approach
- Cleaned and engineered financial ratio features.
- Tested multiple models (logistic regression, random forest, XGBoost).
- Evaluated using precision, recall, ROC-AUC.
- Interpreted model outputs using SHAP.
### AI/ML Considerations
- Class imbalance handling.
- Explainability for risk stakeholders.
- Threshold tuning for operational use.
### Outcome
Achieved strong predictive performance with interpretable insights for decision-makers.
### What I’d improve next
Add time-series features and integrate macroeconomic indicators.
